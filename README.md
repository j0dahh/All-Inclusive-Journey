# All-Inclusive Journey

A 2D educational and accessibility-focused game collection designed for children with disabilities.  
The project aims to create inclusive and engaging experiences through colorful visuals, simple mechanics, and adaptive gameplay.

## Mini-Games
1. **Balloon Popper** – Pop rising balloons to improve reaction time and motor control.
2. **Color Match** – Match colors to enhance visual recognition.
3. **Shape Sorter** – Sort shapes to strengthen cognitive and spatial awareness.
4. **Pair Matching** – Match identical pairs to boost memory and focus.

## Features
- Accessibility-focused interface and visuals
- Simple, rewarding gameplay suitable for all ability levels
- Educational design promoting inclusivity
- Positive reinforcement with sounds and visual effects

## Tools & Technologies
- **Engine:** Unity (2D)
- **Language:** C#
- **Art Style:** Bright, simple, and engaging
- **Development:** Solo – designed, programmed, and integrated all mini-games

## Purpose
Developed as part of a research project exploring gamification for children with disabilities, aiming to make learning more interactive and inclusive.
